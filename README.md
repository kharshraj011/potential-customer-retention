# Customer Retention Analytics Dashboard

An interactive dashboard for exploring customer churn patterns
and retention metrics, built from a Kaggle dataset.

## Live Demo
[View Dashboard](https://potential-customer-retention.lovable.app)

## Features
- Churn rate analysis by segment
- Retention trend visualization
- Interactive filters by region, plan, tenure
- KPI summary cards

## Tech Stack
- React + Vite
- Tailwind CSS
- Recharts / Chart.js
- Data: Kaggle Customer Churn Dataset
## Getting Started
```bash
npm install
npm run dev
```

## Dataset
Downloaded from Kaggle:
[Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
